1. What is the difference between map, filter, foreach, and find?

Ans. map function will return an array based on the condition but foreach function returns undifined. filter method returns an array of matching elements else will return an empty array if no matching happends. Finally, find method did not search the whole array if the condition satisfies and return the single value .

2. What is the difference between local storage, session storage and the cookies?

Ans. Local Storage are saved on the browser, whereas sessions are saved on the server and cookies are saved on the client side. Data is saved on the local storage parmanently, whereas the data on the session storage will disappear by closing the tab.
